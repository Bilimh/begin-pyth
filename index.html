1. Variables et Affichage

Stocker et afficher des informations.


# Programme 2 : Variable simple
prenom = "Alex"
print("Bonjour", prenom)

2. Interaction Utilisateur

Récupérer du texte ou des nombres avec input().


# Programme 3 : Entrée utilisateur
prenom = input("Entre ton prénom : ")
age = int(input("Entre ton âge : ")) # int() pour un nombre
print("Bienvenue", prenom, "tu as", age, "ans")

3. Les Conditions

Prendre des décisions avec if, elif, else.


note = int(input("Entre une note : "))
if note >= 16:
    print("Très bien")
elif note >= 12:
    print("Bien")
else:
    print("À améliorer")

4. Les Boucles "For"

Répéter une action un nombre de fois précis.


# Afficher 5 fois Bonjour
for i in range(5):
    print("Bonjour")

# Parcourir une liste
prenoms = ["Alex", "Samir", "Lina"]
for p in prenoms:
    print(p)

5. Les Boucles "While"

Répéter tant qu'une condition est vraie.


mot = ""
while mot != "stop":
    mot = input("Écris stop pour arrêter : ")

6. Les Fonctions

Créer des blocs de code réutilisables avec def.


def dire_prenom(prenom):
    print("Bonjour", prenom)

dire_prenom("Alex") # Appel de la fonction

7. Les Dictionnaires

Organiser des données par clés et valeurs.


personne = {"prenom": "Alex", "age": 20}
print(personne["prenom"]) # Affiche Alex

9. Créer un Chatbot

Utiliser l'API OpenAI pour discuter avec l'IA.


from openai import OpenAI
client = OpenAI(api_key="TA_CLE_API")

while True:
    question = input("Toi : ")
    response = client.chat.completions.create(
        model="gpt-4.1-mini",
        messages=[{"role": "user", "content": question}]
    )
    print("Bot :", response.choices.message.content)
